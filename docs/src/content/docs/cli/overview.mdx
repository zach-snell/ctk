ctk provides a traditional command-line interface alongside its MCP server mode.
All CLI commands support a `--json` flag for raw JSON output.

## Global Flags

| Flag | Description |
|------|-------------|
| `--json` | Output raw JSON instead of formatted tables |
| `--version` | Print the ctk version |
| `--help` | Show help for any command |

## Output Modes

By default, ctk outputs human-readable formatted tables. Use `--json` for raw JSON:

```bash
# Formatted table output
ctk spaces list

# Raw JSON output
ctk spaces list --json
```
